Downloading & installing PoopWeb

To download PoopWeb, go to the repository and download the latest release.
You will get a zip file including the binaries for PoopWeb. Extract the zip file and run the extracted .exe file inside the folder.

You have to extract the files for PoopWeb to work properly - do not run them inside the zip.

The directory structure

PoopWeb now has generated 3 new folders: conf, log and www. Each of these folder has a purpose.
The conf folder contains configuration files.
The log folder contains the log files, which show which sites were visited, etc.
The www folder is your folder to put in any files which will be available on your site.

Configuration

You can change quite a few settings to adjust PoopWeb to your likings. To edit the configuration, open conf/config.json in your preferred editor. In this file you can change stuff like the HTTP-Port, the MySQL connection credentials, etc.
